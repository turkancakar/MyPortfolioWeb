Projects

Organic Food Web

This project is designed as a modern organic food e-commerce website. Developed using SASS and the BEM methodology, it features a responsive layout with Bootstrap 5.3.3. Google Fonts (Inter, Lato, Rubik) are used, and HTML5 with CSS animations provides a rich user experience. Modern web features such as video backgrounds, form validation, and full device compatibility are also integrated.

The Higher Lower Game

This project is a "Higher or Lower" game based on guessing the follower counts of social media accounts. The player guesses which of the two accounts has more followers. Correct guesses increase the score, while a wrong guess ends the game. Developed with HTML, CSS, and JavaScript, the game offers a modern user experience with a Turkish interface, responsive design, and SweetAlert2 notifications.

Weather Forecast

I used Html Css and Javascript when making this Weather Forecast. I pulled the API from OpenWeather. You can find out the weather by entering the desired city name in the "Enter City" section.
